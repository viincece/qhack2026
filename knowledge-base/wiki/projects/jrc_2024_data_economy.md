# JRC/2024/OP/0019 -- Data Economy Monitoring Tool Phase III

## Summary

Phase III of the European Data Market Monitoring Tool, commissioned by the Joint Research Centre (JRC). This ongoing project enhances the NLP pipeline and indicator derivation methodology, aligns with the Data Act and Data Governance Act (DGA), and introduces a new API delivery mechanism for near-real-time data access. It is MIG's largest single contract at EUR 620,000 over 24 months.

## Deliverables

1. **Redesigned web indicator methodology**: Enhanced NLP pipeline with improved indicator derivation aligned to Data Act and DGA
2. **Quarterly dataset**: Structured data deliveries on a quarterly cycle
3. **API wrapper**: New programmatic access mechanism for near-real-time data consumption
4. **Annual trend report**: Comprehensive yearly analysis of European data economy trends

## Technical Approach

The project combines three workstreams:

- **Web data infrastructure**: Leveraging the [[methodology/webmap_pipeline]] with enhanced crawling and classification, targeting data economy entities (classifier F1: 0.88, see [[team/marcus_weber]])
- **Survey design**: Complementing web-derived data with structured survey instruments
- **Stakeholder engagement**: Policy-aligned consultation to ensure indicator relevance, led by [[team/sofia_chen]]

Key innovation in Phase III is the redesigned indicator derivation methodology that maps directly to Data Act and DGA definitions, plus the API delivery mechanism replacing batch-only delivery.
