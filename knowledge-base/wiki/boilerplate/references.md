## Usage Notes
- Always confirm with the contact that they are willing to serve as a reference before including them in a tender submission
- Select references most relevant to the scope of the new tender
- For cybersecurity tenders: prioritise ENISA reference
- For financial regulation tenders: prioritise EBA reference
- For data/digital economy tenders: prioritise JRC reference
- For industrial/manufacturing tenders: prioritise DG GROW reference
