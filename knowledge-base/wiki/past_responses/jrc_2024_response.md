# Past Response: JRC/2024/OP/0019 — European Data Economy Monitoring Tool Phase III

## Client
European Commission — Joint Research Centre (JRC), Seville

## Executive Summary (as submitted)
Meridian proposes Phase III of the European Data Market Monitoring Tool (EDM), building on established track record in web-based organisational data collection and direct EDM methodology experience from previous phases. Enhanced NLP pipeline capabilities, redesigned indicator derivation framework aligned with European Data Act and Data Governance Act, and new API delivery mechanism enabling near-real-time access to indicator updates for JRC staff and Commission policy teams. Combines large-scale web data infrastructure with targeted survey design (subcontracted to survey specialist) and structured stakeholder engagement.

## Problem Framing (reusable pattern)
- Phase III faces two simultaneous challenges: (1) updating the empirical baseline (data economy has changed since Phase II), and (2) extending methodology to capture developments Phase II missed
- New scope includes: data intermediaries under DGA, cloud and edge computing markets
- Need for more granular, more timely, and more actionable monitoring tool
- Meridian brings continuity (previous phase involvement) plus enhanced technical capability

## Methodology Proposed
- [[webmap_pipeline]] with data-economy-specific configuration
- Redesigned web-based indicator derivation: methodology led by [[anna_becker]]
- AI involvement indicator: continuous score for AI-product-or-service relevance (built by [[marcus_weber]], validated against 4,200 manually annotated companies)
- API wrapper for near-real-time indicator access (built by [[thomas_vogel]])
- Subcontracted survey component + structured stakeholder engagement
